Mi Catálogo de Productos
Una aplicación móvil que construí desde cero y que funciona como una tienda real.

¿Qué es esto?
Esta es mi aplicación móvil completa, desarrollada con React Native y TypeScript, que funciona como un catálogo de productos profesional. No es un ejercicio: es una app que funciona de verdad, con login, carrito, navegación y todo lo que tiene una tienda real.

¿Qué podés hacer con mi app?
- Iniciar sesión con tu email y contraseña (como cualquier página)
- Agregar productos al carrito con +, −, total y eliminar
- Ver una pantalla de bienvenida después de login (con tu nombre)
- Agregar productos desde un formulario profesional
- Finalizar la compra con resumen y mensaje de éxito
- Todo como una página real (como Musimundo, pero hecha por mí)

¿Cómo la construí?
La construí paso a paso:
1. Primero hice el login, con email y contraseña real
2. Después hice el carrito, con +, −, total y eliminar
3. Después hice el formulario de alta, con el botón abajo
4. Después hice la pantalla de bienvenida, con mi nombre
5. Después hice el botón de finalizar compra, con resumen y éxito
6. Y todo funcionando sin errores, como una página real

Tecnologías que usé (y aprendí)
- React Native: para la interfaz móvil
- TypeScript: para que nada se rompa
- React Navigation: para que todo fluya
- Expo: para desarrollar y probar en mi celular
- GitHub: para que todo esté versionado y profesional

¿Cómo probás mi app?
1. Cloná el repositorio
2. Instalá las dependencias
3. Corré el servidor
4. ¡Listo! La app funciona en tu celular

git clone https://github.com/chericastaa/CatalogoProductos.git
